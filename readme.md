# HadeanTacticsTemplate

An example mod for various objects modder may want to make or modify. Mods for [Hadean Tactics](https://store.steampowered.com/app/1324530/Hadean_Tactics/).

## Prerequisites

- [.NET SDK](https://dotnet.microsoft.com/download)
- An owned copy of Hadean Tactics (Steam)
- [BepInEx 5](https://docs.bepinex.dev/) installed in the game folder
